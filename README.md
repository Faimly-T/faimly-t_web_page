# Faimly-T Web Page

Premium landing page for Faimly-T AI Software & Consultancy.

## Local Development
Simply open `index.html` in any browser or run a local web server.

## Deployment
This website is designed for zero-cost static hosting (e.g., GitHub Pages).
To host on GitHub Pages:
1. Ensure the repository name is `faimly-t_web_page`.
2. Go to **Settings** > **Pages** on GitHub.
3. Set Build and deployment source to **Deploy from a branch** -> `main` branch -> `/ (root)`.
4. Click **Save**.
5. Add your custom domain under the Custom domain section.
